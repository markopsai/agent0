## How to Use This File

Tell the AI to update this file after every major code change and to read it before every major code change. Include:
- The date of the change
- What you changed
- Why you changed it (optional)

## Format Example
```
### [Date] - Feature/Fix Name
- **Changed**: Description of what you modified
- **Files**: List of files you edited
- **Reason**: Why you made this change (optional)
```
